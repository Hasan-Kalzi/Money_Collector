# *Money Grabber*

A simple 2D game where  you controll the character by touching the screen and thus making it move up or down.
The goal is to grab coins without getting hit by rockets..

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

What things you need to install the software and how to install them

* [Android Studio](https://developer.android.com/studio "DOWNLOAD ANDROID STUDIO")

* An android device with at least android 5.1 ( [Android Lollipop](https://en.wikipedia.org/wiki/Android_Lollipop "Android Lollipop Wikipedia" ) )

**OR**

* Create a virtual device inside android studio (Using Android Virtual Device Manger) : Follow the instructions below.
### Installing

A step by step series of HasanKalzis that tell you how to get a development env running

Say what the step will be

* Step 1
```
Downlowd Android Studio and install it follow install instruction
```

* Step 2

```
Click on the green button up corner "Clone or Download" and choose Download ZIP
```
* Step 3
```
Unzip the file in the desired location
```
* Step 4
```
Open Andoid Studio and click on "Open Project" and the folder where you fetched the zip 
```
* Step 5
```
Connect your mobile on USB and activate USB Debugging
```
[USB Debuggning](https://developer.android.com/studio/debug/dev-options "Configure on-device developer options")

**OR**
```
Create a virtual device on android studio
```
[Create an emulator for testing in Android Studio](https://developers.foxitsoftware.com/kb/article/create-an-emulator-for-testing-in-android-studio/ "Create an emulator for testing in Android Studio")

* Step 6
```
Press on green play button and enjoy!
```
